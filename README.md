v1 – Released on 14/01/2026
	The design and features are nearly identical, with only minor modifications required.
	The extension includes 3 layers:

		Layer 1 – Header (Chrome System):
			Bookmark: Access your Chrome bookmarks.
			Application: Apps installable from the Chrome Web Store.
			History: Browsing history.
			Download: Downloads.

		Layer 2 – Main (Shortcuts):
			Personal shortcuts:
				Amazon
				Wikipedia
				Google
				Facebook
				YouTube
				X (formerly Twitter)
				Instagram
				Reddit


		Layer 3:
			Columns containing all shortcuts and the full interface.
